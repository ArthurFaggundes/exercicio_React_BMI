# 📌 BMI Calculator

Responsive BMI (Body Mass Index) Calculator web application built with React + Vite, focused on practicing components, state management, and form interactions.

🌐 Access the project:
[Project Website](https://exercicio-react-bmi.vercel.app/)

## 📖 About the Project

This project was developed during my React studies with the goal of practicing fundamental Front-End development concepts such as component-based architecture, state management, and dynamic rendering.

The application allows users to enter their weight and height to automatically calculate their Body Mass Index (BMI), displaying the result in a simple and intuitive way.

## ✨ Features

* Automatic BMI calculation.
* Weight and height input fields.
* Display of the calculated result with color spectrum.
* Responsive interface.
* Dynamic updates using React.

## 🛠️ Technologies Used

    - React
    - JavaScript
    - HTML5
    - CSS3
- Tools:
    - Vite
    - GitHub
    - Vercel

## 🎯 Purpose

This project was created to:

* Practice core React concepts;
* Learn state management with functional components;
* Build responsive user interfaces;
* Improve modern JavaScript skills.

## 📖 Sobre o Projeto

Este projeto foi desenvolvido durante meus estudos de React com o objetivo de praticar conceitos fundamentais do desenvolvimento Front-End, como componentização, manipulação de estados e renderização dinâmica de informações.

A aplicação permite que o usuário informe seu peso e altura para calcular automaticamente o Índice de Massa Corporal (IMC), retornando o resultado de forma clara e intuitiva.

## ✨ Funcionalidades

* Cálculo automático do IMC.
* Campos para inserção de peso e altura.
* Exibição do resultado calculado juntamente com mudança de cor conforme resultado.

* Atualização dinâmica das informações utilizando React.
